# trangcanhan
# 📖 Giới thiệu dự án

Website được thiết kế như một portfolio cá nhân giúp:

- Giới thiệu thông tin cá nhân
- Trình bày kỹ năng công nghệ
- Showcase các dự án đã thực hiện
- Cung cấp thông tin liên hệ
- Tạo dấu ấn cá nhân trên môi trường số

Dự án phù hợp để:
- Làm CV online
- Portfolio xin việc
- Thực hành Frontend
- Học HTML/CSS/JavaScript
- Deploy website bằng GitHub Pages hoặc Vercel
